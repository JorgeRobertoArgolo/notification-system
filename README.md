# Atividade Prática: Padrões GoF

## Questão 1. Aplicação do Padrão Factory Method no Desenvolvimento de um Sistema de Notificações.

### Cenário Fictício

Você foi contratado para desenvolver um sistema de notificações para uma empresa que
precisa enviar diferentes tipos de notificações aos seus clientes, como e-mails, SMS e
notificações via push. 

A ideia é que o sistema seja flexível para permitir a adição de novos tipos de notificações no futuro, sem precisar alterar o código já existente. Para isso, a empresa solicita que o *desenvolvimento siga o padrão de projeto Factory Method* para a criação dos diferentes tipos de notificações.

### Objetivo

Implementar uma estrutura que utilize o padrão Factory Method para criar e enviar diferentes
tipos de notificações. Os alunos deverão criar as classes necessárias para seguir o padrão e
simular o envio de notificações utilizando diferentes meios.
